PASO 1 (solo una vez)
─────────────────────
  Doble clic en:

      ABRIR GOS Objetivos.bat

  La primera vez crea el archivo .env con secretos seguros.


PASO 2 (cada vez que quieras usar el sistema)
──────────────────────────────────────────────
  Doble clic en:

      ABRIR GOS Objetivos.bat
      o el icono del Escritorio (si lo creaste)

  Se abre el navegador automaticamente (Chrome, Edge, etc.).


PASO 3 - Ingresar
─────────────────
  En desarrollo local (GOS_AUTO_LOGIN=true):
    Entras directo sin login.

  Si pedis login manual:
    Email:    admin@demo.local
    Clave:    ver GOS_ADMIN_PASSWORD en el archivo .env


IMPORTANTE
──────────
  • El servidor corre en segundo plano.

  • Para apagar: Administrador de tareas → finalizar "pythonw"
    o reiniciar la PC.

  • Verificar configuracion: VERIFICAR ENV.bat


SI NO ABRE EL NAVEGADOR
───────────────────────
  Escribe en el navegador:

      http://127.0.0.1:5000


SI DICE QUE NO HAY PYTHON
─────────────────────────
  Instala desde: https://www.python.org/downloads/
  Marca: [x] Add python.exe to PATH
  Reinicia la PC y vuelve a intentar.
